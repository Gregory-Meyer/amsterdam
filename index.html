libamsterdam

libamsterdam is an implementation of asynchronous channels for C++17.

Usage

1 {c++}
2 #include <cassert>
3 
4 #include <amsterdam.h>
5 
6 int main() {
7  auto [tx, rx] = amst::channel<int>();
8 
9  tx.push(5);
10  assert(tx.pop() == 5);
11 }

Installation

```sh git clone https://github.com/Gregory-Meyer/amsterdam.git mkdir amsterdam/build cd amsterdam/build cmake .. -DCMAKE_BUILD_TYPE=Release cmake –build . -j nproc cmake –build . –target test sudo cmake –build . –target install ```
